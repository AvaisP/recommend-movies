Movie Recommeder System

This is a movie recommender system built on django using the K-means clustering algorithm.It suggests movies that the user must watch next based on what movies similar users have liked.

Getting Started Source on GitHub
To start using the recommender system click on the Getting Started button.

How this system works

  • This app takes user id as an input.
  • It then locates the clusters the user belongs to.
  • We then find similar users present in the cluster.
  • This is followed by recommending movies to the user from the movies rated by similar users present in the cluster.

Issues

  • Right now, the app uses the K-means clustering algorithm.
  • In the future we would like to move on to a collaborative filtering algorithm for item-item similarity using the cosine similarity function.
  • It doesn't let new users join. That is there is no way to recommend movies to a user without a history.
  • It doesn't work on real time data.
Please keep the issues in mind before using the app. You can work on the issues and submit a fork.
